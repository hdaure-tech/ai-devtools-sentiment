# Contributing to AI DevTools Sentiment

We welcome tool requests, data corrections, and methodology feedback.

## How to contribute

- Open a GitHub Issue using the `🔍 Request a Tool Analysis` template to suggest a tool.
- Open a `📊 Data Correction` issue if a score, complaint, praise, or source looks off.
- Use Issues for methodology feedback if you think a source or weighting rule should change.

Murmure runs weekly analysis cycles. New tools get added based on demand.
