# Contributing & Requesting a Tool Analysis

## 🔍 Request a Tool Analysis

Is your AI devtool not in the tracker yet? **Open an issue using the "Request Analysis" template.**

We prioritize tools with:
- Active Reddit/HN communities
- 1,000+ GitHub stars or significant developer usage
- Ongoing public discussions about quality, pricing, or reliability

👉 [Open a Request Issue](https://github.com/murmurecc/ai-devtools-sentiment/issues/new?template=request_analysis.md)

## 📊 Contributing Data

If you have community data (Reddit threads, HN posts) you'd like to contribute for a specific tool, open a PR or issue with:
- Tool name
- Source URLs
- Date range
